# Bitcoin-Price-Forecasting-using-Multivariate-Multi-Horizon-Deep-Learning

Proyek ini mengimplementasikan model **Multivariate Multi-Horizon Time Series Forecasting** untuk memprediksi harga Bitcoin menggunakan pendekatan **LSTM** dan **Seq2Seq LSTM**. Pipeline mencakup eksplorasi data, preprocessing, feature engineering, pembangunan arsitektur deep learning kustom, custom training loop dengan TensorFlow, serta evaluasi performa model untuk prediksi 24 langkah ke depan.
